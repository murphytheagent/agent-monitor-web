Tokenizer Observatory

An interactive, client-side tokenizer view for mainstream open-source LLM prompts. Switch between public instruct tokenizers and inspect both raw text and chat-template serialization in the browser.

Loading tokenizer

First load pulls tokenizer assets from Hugging Face, then the browser cache takes over.

What this page does: it shows exactly how input text is chopped into token IDs, what each sub-token looks like in the underlying vocabulary, and how the selected model changes the final prompt before generation.

Why the hosted route is separate: the main dashboard homepage is auto-regenerated by the monitor publisher, so this tokenizer lives at its own stable Pages path and won’t get overwritten by snapshot refreshes.

What to inspect

  • How multilingual text, code, whitespace, and emojis fragment into token pieces.
  • How many tokens each chat template spends before any generation begins.

Hosted Constraint

This static route only includes tokenizer repos that can be fetched directly from a public browser session. Gated families such as Meta Llama and Google Gemma need authenticated access or a backend proxy, so they are not wired into this build yet.
